// Authenticated と並行して保持。Supabase Realtime で status 変化を監視

04 — ルームリスト
// RoomListFeature (iOS · TCA) · RoomListViewModel (Android · MVVM)
⚑ Global Generation（03）と並行して動作
stateDiagram-v2 [*] --> loading loading --> idle : ロード完了 loading --> loadFailed : ロード失敗 loadFailed --> loading : リトライ loadFailed --> error : リトライ失敗 error --> loading : 再試行（ロード）を選択 error --> idle : キャンセルを選択 idle --> requesting : ユーザーアクション requesting --> idle : 通信成功 requesting --> idle : 通信失敗 [AppError] idle --> requesting : 手動リトライ

// AppError — エラー種別定義

networkError ネットワーク障害 / タイムアウト
usageLimitExceeded 無料プランの生成回数上限超過（サーバー側で判定・返却）
serverError(code: Int) サーバーエラー（5xx 系）
unauthorized 認証切れ → ログインモーダルへ誘導

06 — AI生成画面
// GenerateFeature (iOS · TCA) · GenerateViewModel (Android · MVVM)

// 通信成功後の副作用

image_contents.status = "generating" で DB レコード作成済み（Edge Function が更新）
Global Generation notGenerating → generating に遷移
usageLimitExceeded AppError として返却 → idle に戻りエラー表示
